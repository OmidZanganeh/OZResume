# Archive

Use this folder for files you want to **keep locally** but **exclude from Cursor** (indexing and default context).

**Examples:** large GeoJSON/KML exports, screenshots, design dumps, zip backups, one-off data files, old assets you are not actively editing.

- Excluded via **`.cursorignore`** (`archive/`).
- **Git:** If these should not be committed, add `archive/*` (or specific patterns) to **`.gitignore`** as well. This README can stay tracked; ignore only the bulky contents you add.
