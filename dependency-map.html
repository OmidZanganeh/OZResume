🛠GIS tools by route

Each tool is its own mini‑app with a page, CSS, optional layout, and APIs.

Coordinate Converter /tools/coordinate-converter

Converts coordinates between DD, DMS, DDM and syncs with a Leaflet map.

  • UI: page.tsx + page.module.css
  • Map: MapPicker.tsx + MapPicker.module.css (Leaflet, OpenStreetMap).
  • SEO: layout.tsx with WebApplication JSON‑LD.
Spatial Unit Converter /tools/unit-converter

Pure client‑side conversions between distance, area, angle units.

  • UI: page.tsx + page.module.css
  • SEO: layout.tsx (WebApplication schema).
File Tools (CSV → GeoJSON & EXIF) /tools/image-tools

Converts CSV with lat/lon to GeoJSON and reads EXIF GPS from photos.

  • UI: page.tsx + page.module.css
  • EXIF: uses exifr (client-side parser).
  • SEO: layout.tsx with JSON‑LD.
Isochrone Mapper + POI Finder /tools/isochrone

Generates travel-time polygons from any origin, then finds POIs (restaurants, gas, etc.) inside a chosen ring.

  • UI: page.tsx + page.module.css
  • Map & polygons: IsochroneMap.tsx + IsochroneMap.module.css
  • Travel‑time API: /api/isochrone→ Valhalla Isochrone (OSM)
  • POI search: /api/overpass→ Overpass API (OSM POIs)
  • SEO: layout.tsx (WebApplication schema).
Elevation Profile Tool (USGS 3DEP) /tools/elevation-profile

Samples points every N feet along a drawn path, queries USGS 3DEP, and renders an SVG elevation chart.

  • UI: page.tsx + page.module.css
  • Map & path drawing: ElevationMap.tsx + ElevationMap.module.css
  • Elevation API: /api/elevation→ USGS EPQS v1 (3DEP)
    Retries transient errors; never interpolates, just skips true no‑data.
  • SVG chart + CSV/SVG downloads implemented directly in page.tsx.
  • SEO: layout.tsx (WebApplication schema).
